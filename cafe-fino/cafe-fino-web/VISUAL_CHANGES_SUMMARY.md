# Café Fino — Visual Changes Summary

## Before → After Comparisons

1) **Navbar**
- **Before:** Standard left-aligned logo, horizontal links
- **After:** Fully centered layout, stacked sections (lang toggle → nav → locations), sticky with shadow

2) **Hero/Intro**
- **Before:** Static logo fade-in with steam effect
- **After:** 3-second image cascade (8 images fade → compress → reposition behind logo)

3) **Color Palette**
- **Before:** Espresso/Mocha/Cream/Gold (generic, similar to Le Clay)
- **After:** Deep Roast Black/Cream/Crema Gold (coffee-bean inspired, distinctive)

4) **Menu Items**
- **Before:** Static cards, no hover feedback
- **After:** Lift on hover, shadow expand, image scale, gold underline slide

5) **Buttons**
- **Before:** Static CTA buttons
- **After:** Scale on hover, color shift (Crema Gold → Coffee Brown), shadow expand

6) **Overall Feel**
- **Before:** Functional, basic
- **After:** Premium, engaging, cohesive brand experience
