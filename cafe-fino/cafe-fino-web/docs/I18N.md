# i18n Guide

## Where It Lives

- Init: [src/i18n.js](../src/i18n.js)
- Locales: [src/locales](../src/locales)

## Conventions

- Default language is French; English is secondary.
- Translation keys are grouped by feature: nav, hero, menu, clickCollect, locations, instagram, footer, slider.
- Arrays are returned via returnObjects when needed (see menu.brunch.includes).

## Adding Keys

1. Add key to both [src/locales/fr/translation.json](../src/locales/fr/translation.json) and [src/locales/en/translation.json](../src/locales/en/translation.json).
2. Use descriptive ids for menu items that match menu.json ids.
3. Keep placeholders consistent for interpolations.

## Examples

- Plain string:
  - nav.menu
- Object:
  - menu.categories.breakfast
- Array:
  - menu.brunch.includes

## Language Persistence

- Language is stored in localStorage under cafe-fino-lang.
